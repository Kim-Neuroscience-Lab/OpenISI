# Changelog

All notable changes to OpenISI will be documented in this file.

## [Unreleased]

### Added
- Initial open-source release
- VSync-locked stimulus presentation (drifting bars, rotating wedges, expanding rings)
- Multi-camera support (pco.panda, AVFoundation, OpenCV)
- Real-time acquisition metrics and timing validation
- Frame-accurate hardware timestamp support
- Automated display refresh rate validation
- Cross-platform support (macOS, Windows, Linux)
- Self-contained distribution with embedded Python runtime
- In-app update checking via GitHub Releases
